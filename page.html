Bienvenue dans notre projet Git

Cette page est conçue pour vous aider à pratiquer Git en équipe. Vous pouvez créer des branches, faire des commits, résoudre des conflits et fusionner vos travaux.

Créer des branches

Chaque membre de l'équipe peut créer sa propre branche pour travailler sur des fonctionnalités spécifiques sans affecter la branche principale.

Faire des commits

Enregistrez vos modifications avec des messages de commit descriptifs pour suivre l'évolution du projet.

Fusionner les branches

Une fois vos fonctionnalités terminées, fusionnez vos branches dans la branche principale pour intégrer vos travaux.

À propos du projet

Ce projet a pour objectif de vous familiariser avec les concepts de base de Git en situation de collaboration. Vous pouvez modifier n'importe quelle partie de cette page pour pratiquer.

Objectifs d'apprentissage

  • Créer et gérer des branches
  • Résoudre les conflits de fusion
  • Utiliser les pull requests
  • Collaborer efficacement avec Git

Instructions

Chaque personne devrait créer sa propre branche, apporter des modifications, puis essayer de fusionner avec la branche principale. Vous pouvez également créer des conflits intentionnellement pour pratiquer leur résolution.
